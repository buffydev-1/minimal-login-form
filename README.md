# 🔐 Minimal & Responsive Auth System

A clean, modern, and fully responsive Authentication System built with pure **HTML5**, **CSS3**, and **JavaScript**. This project features a seamless toggle between Login and Sign Up views with smooth animations and interactive elements.

## ✨ Features

- 📱 **Fully Responsive:** Adapts seamlessly to all screen sizes.
- 🔄 **Smooth View Switching:** Instant toggle between Login and Sign Up with CSS transitions.
- 👁️ **Password Visibility Toggle:** Show/Hide password functionality for better UX.
- ⏳ **Dynamic Loading States:** Buttons with mock loading indicators on submission.
- 🎨 **Modern UI:** Clean aesthetic using the 'Inter' font family and a professional color palette.
- 🏗️ **Accessible & Clean:** Semantic HTML and well-organized CSS/JS.

## 🛠️ Built With

- **HTML5:** Semantic structure for Login and Sign Up forms.
- **CSS3:** Flexbox, Custom Properties (Variables), Media Queries, and Transitions.
- **JavaScript (Vanilla):** View switching, password toggles, and submission animations.
- **Google Fonts:** [Inter](https://fonts.google.com/specimen/Inter) for typography.

## 📂 Project Structure

- index.html: Main HTML file containing both Login and Sign Up forms.
- style.css: Modern styling and layout rules.
- script.js: Interactive functionality and UI logic.
- README.md: Project documentation.
